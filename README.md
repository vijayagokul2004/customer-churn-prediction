# 📊 Customer Churn Prediction – Machine Learning Project

## 📌 Problem Statement
Telecom companies lose a significant amount of revenue due to customers discontinuing their services. The goal of this project is to predict whether a customer will churn or not based on historical usage behavior, account details, and service patterns.

---

## ⚙️ Approach / Methodology

### 1. Data Collection & Understanding
- Loaded customer dataset containing demographics and service usage details
- Studied distribution of churn vs non-churn customers

### 2. Data Preprocessing
- Handled missing values
- Encoded categorical variables
- Scaled numerical features where required

### 3. Exploratory Data Analysis (EDA)
- Identified key factors influencing churn
- Analyzed customer behavior patterns

### 4. Model Building
- Implemented Random Forest Classifier
- Split dataset into training and testing sets
- Trained model on processed features

### 5. Model Evaluation
- Accuracy
- Precision
- Recall
- F1 Score

---

## 📊 Results
- Accuracy: 98.34%
- Precision: 98.34%
- Recall: 100%
- F1 Score: 99.16%

---

## 💡 Business Impact
- Helps telecom companies identify customers at risk of leaving
- Enables targeted retention strategies
- Reduces customer loss and increases revenue stability
- Improves customer satisfaction through proactive engagement
